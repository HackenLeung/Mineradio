## Mineradio 1.1.2 更新内容

1.1.2 主要整理评论区、智能过渡和继续听体验，主逻辑集中在 `public/index.html`：

- **评论区：** 红心按钮右侧新增评论按钮，显示评论数量，最多显示为 `999+`。
- **评论面板：** 点击评论按钮打开评论面板，支持网易云音乐和 QQ 音乐评论接口。
- **智能过渡：** 歌曲切换新增智能过渡效果，底部显示“智能过渡”标识。
- **继续听：** 优先接上上次恢复的播放队列和当前歌曲。
- **Home：** 显示当前恢复队列里的歌曲、封面、歌手和队列位置。
- **恢复播放：** 点击“继续听”会从恢复队列当前位置继续播放，不再新建单曲队列。
- **其他修正：** 点击歌曲详情时保持二级面板逻辑，Home 卡片部分英文标签改为中文。

> **已知问题：**
>
> - 第三方接口或账号状态不可用时，在线搜索、歌单导入、歌词补全或在线播放可能失败。
> - QQ 音乐播放依赖真实播放票据；只有网页账号态时可能只能读取资料，不能直接播放。
> - 桌面歌词、壁纸模式和透明窗口仍受 Windows、显卡驱动、桌面软件和安全软件影响。

## 开发运行

```bash
npm install
npm start
npm run build:win
```

桌面版入口由 Electron 主进程加载本地服务。`npm run build:win` 会生成 Windows 构建产物，产物位于 `dist/`。

常用检查：

```bash
git diff --check
node --check server.js
```
