# Mineradio

Mineradio 是 Windows Electron 桌面音乐播放器，包含在线搜索与播放、本地音乐库、歌词舞台、粒子视觉、3D 歌单架、音频调节和自动更新。

## Mineradio 1.1.5 更新内容

- **平台展示：** 在线平台在界面和文档中统一显示为小云、小Q、小狗，内部兼容协议和账号数据格式保持不变。
- **性能优化：** 低/中/高画质分别限制为 30/45/60 FPS，超高画质保留高刷；失焦降到 24 FPS，最小化降到 1 FPS。
- **启动优化：** 恢复 Chromium 后台节流，本地库扫描、玻璃纹理计算和 WebGL 预编译改为启动页退出后错峰执行。
- **三平台账号：** 小云、小Q、小狗登录状态可同时保留；默认聚焦当前平台，“全部”用于临时汇总，重启后恢复选择。
- **小狗：** 新增二维码/网页/Cookie 登录、账号与歌单读取、歌单详情、歌曲搜索、播放地址、歌词和多档音质支持。
- **统一搜索：** 搜索栏新增 `KG`；`All` 聚合小云、小Q、小狗，并优先排列当前账号平台结果。
- **本地音乐库：** 支持选择文件夹、按文件夹生成二级歌单、读取同目录封面和歌词、搜索本地歌曲，并在重启后重新绑定本地文件继续播放。
- **平台歌单导入：** 支持小Q、小云、酷我、小狗和咪咕公开歌单，导入结果保存在本地。
- **Home：** 增加紧凑歌单卡片、可排序歌单区、自定义首页图片和文案；天气仅作为可选信息显示，定位和城市设置集中在“自定义首页”。
- **桌面体验：** 增加系统托盘播放控制、关闭到托盘、开机启动、沉浸自动全屏和本地文件夹扫描。
- **播放与视觉：** 增加倍速/音调、均衡器、局内歌词行数与竖排、壁纸镜像及 Wallpaper Engine 素材浏览。
- **稳定性：** 修复本地歌曲恢复播放、平台切换残留歌单详情、缺失天气显示为 `0°` 等问题；天气获取失败时不显示天气区域。

详细记录见 [2026-07-10 开发更新与提交审计](./docs/DEVELOPMENT_UPDATE_2026-07-10.md)。

## v1.1.3 基线

- 智能过渡画面支持镜面、频谱、方块、右滑、随机和关闭。
- 评论面板支持小云和小Q评论接口。
- “继续听”优先恢复上次队列、当前歌曲和播放位置。

> **已知问题：**
>
> - 第三方接口或账号状态不可用时，在线搜索、歌单导入、歌词补全或在线播放可能失败。
> - 小Q播放依赖真实播放票据；只有网页账号态时可能只能读取资料，不能直接播放。
> - 桌面歌词、壁纸模式和透明窗口仍受 Windows、显卡驱动、桌面软件和安全软件影响。

## 开发运行

```bash
npm install
npm start
npm run build:win
```

桌面版入口由 Electron 主进程加载本地服务。`npm run build:win` 会生成 Windows 构建产物，产物位于 `dist/`。

常用检查：

```bash
git diff --check
node --check server.js
```
